# Currency Converter Pro

## Description
Currency Converter Pro is a dynamic and user-friendly web application that allows users to seamlessly convert between various global currencies. It leverages live exchange rate data to ensure accurate and up-to-date conversions.

Key features include:
- **Single Conversion**: Quickly convert a specific amount from one currency to another.
- **Multi-Currency Conversion**: Set a base currency and view its equivalent value across multiple target currencies simultaneously.
- **Favorites**: Save frequently used currency pairs for quick access.
- **Interactive UI**: Features currency flags, a responsive design, and an intuitive currency picker.
- **Dark Mode**: Built-in theme toggle for enhanced user experience.

## Requirements
- A modern web browser (e.g., Google Chrome, Mozilla Firefox, Apple Safari, Microsoft Edge).
- **Internet Connection**: Required to fetch real-time exchange rates from the external API (ExchangeRate-API).
- No installation or backend server required. This is a purely frontend application built with HTML, CSS, and JavaScript.

## How to Run
Simply open the `index.html` file in your preferred web browser. Ensure you have an active internet connection so the application can fetch the latest currency exchange rates.
