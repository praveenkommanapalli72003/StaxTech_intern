# Password Generator Pro

## Description
Password Generator Pro is a feature-rich, web-based utility designed to generate highly secure and customizable passwords, passphrases, and PINs. It provides an intuitive interface with options to control password length, character sets (uppercase, lowercase, numbers, symbols), and exclude ambiguous characters. 

Key features include:
- **Password Generation**: Create standard passwords with specific constraints.
- **Passphrase Generation**: Generate memorable passphrases with customizable word counts and separators.
- **PIN Generation**: Create secure numerical PINs with options to avoid repeated or sequential digits.
- **Bulk Generation**: Generate multiple passwords at once.
- **Strength Estimation**: Real-time password strength and entropy calculation.
- **History Management**: Save, export, and import generated passwords locally.
- **Dark Mode**: Toggleable dark theme for better visibility.

## Requirements
- A modern web browser (e.g., Google Chrome, Mozilla Firefox, Apple Safari, Microsoft Edge).
- No installation or backend server required. This is a purely frontend application built with HTML, CSS, and JavaScript.

## How to Run
Simply open the `index.html` file in your preferred web browser to start using the Password Generator Pro.
